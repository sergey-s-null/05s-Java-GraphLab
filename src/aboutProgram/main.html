Меню программы

  • Файл
    • Новый граф. Создает новую вкладку с пустым графом.
    • Открыть. Открывает граф в одном из следующих форматов: "*.adj", "*.inc", "*.ed".
    • Сохранить. Сохраняет граф в одном из следующих форматов: "*.adj", "*.inc", "*.ed".
    • Сохранить изображение. Сохраняет изображение графа в формате "*.png".
  • Редактировать
    • Отмена. Отменяет последнее изменение графа.
      • Отменяемые действия:
      • Добавление/удаление вершины/ребра
      • Перемещение вершины/ребра
      • Изменение имени вершины
      • Изменение веса/направления ребра
      • Изменение ширины/высоты поля графа
    • Вернуть. Возвращает изменение графа.
    • Размер поля. Меняет ширину и высоту поля графа.
  • Задачи теории графов
  • ?
    • О программе. Вы здесь.
    • Об авторе. Информация об авторе программы.

Форматы файлов

Все файлы представляют из себя набор групп типа group_name{group_content};.
Пример: Matrix{[0, 2], [3.3, 4]};
Далее приведены группы для каждого формата.
* обозначает необязательную группу.
  • Матрица смежности (*.adj).
    • Matrix. Содержит матрицу смежности в формате [a, b, c], [d, e, f],..., где [] - строка матрицы. Матрица строго квадратная.
    • *Vertices. Содержит информацию об именах и расположении вершин: n1(x1, y1), n2(x2, y2),..., где ni - имя вершины, xi, yi - координаты. Кол-во вершин совпадает с размером матрицы.
    • *Resolution. Содержит width, height - размер поля графа.
  • Матрица инцидентности (*.inc).
    • Matrix. Содержит матрицу инцидентности графа. Аналогично "*.adj": [...], [...],...
    • *Edges. Содержит информацию о ребрах в формате: (a1, b1), (a2, b2),...
      В случае бинарного ребра a - угол отклонения вектора вершины ребра относительно вектора проведенного от первой вершины до второй. b - отношение расстояния между вершиной ребра и первой вершиной к расстоянию между первой и второй вершинами.
      Для унарного ребра (a, b) - вектор, проведенный от вершины графа до вершины ребра.
    • *Vertices. Аналогично "*.adj". Кол-во вершин совпадает с кол-вом строк в матрице инцидентности.
    • *Resolution. Аналогично "*.adj".
  • Ребра (*.ed).
    • Edges. Содержит информацию о ребрах графа в формате: (...), (...),...
      В случае унарного ребра скобка выглядит следующим образом: (w, n, x, y), где w - вес ребра, n - имя вершины графа, x и y - аналогично (a, b) из "*.inc" Edges.
      В случае бинарного ребра скобка выглядит следующим образом: (w, n1, n2, d, a, r), где w - вес ребра, n1 и n2 - имена вершин, d - направление ребра(-1 - к первой вершине, 1 - ко второй, 0 - ненаправлено), a и r - аналогично "*.inc" Edges.
    • *Vertices. Аналогично "*.adj". Имена вершин из определения Edges должны быть подмножеством имен из определения Vertices.
    • *Resolution. Аналогично "*.adj"
